# Math-behind-Signal-and-System

## 简介

这个仓库存储了《信号与系统中的数学》的所有代码，可以找到最新版的pdf进行下载。本书正在持续更新中，预计2026年3月前完成大部分内容，在之后的半年里做一些微调。

本书旨在对信号与系统的数学原理进行补充和深化，大部分内容取材于“傅里叶分析”课程，并且融合了一些工程上的应用，以及常微分方程、实变函数、复变函数和泛函分析等课程的初步知识。

## 用法

对于数学基础比较好的读者，可以在课余时间阅读本书，补充课程中略去的数学内容；对于数学基础一般的读者，阅读本书的部分小节可能比较困难，可以先学习“信号与系统”课程，在遇到教材没有解释清楚的内容时，再参考本书。

读者可以跳过一些不感兴趣的内容，对于严谨性没有太高要求，仅希望学会形式计算时，跳过一些小节不会有问题。书中同样为重要程度相对较低，或者难度较高的小节标注了"*"，但读者最好浏览这些小节中的主要结论。

## 特色

首先，本书并排给出了频率形式、角频率形式的傅里叶变换，方便读者对比两种公式及其证明。在讨论特定问题时，选取其中一种傅里叶变换，会使得内容更加简介明快。

其次，就信号与系统本身的内容而言，本书也在讲解的思路上进行了一些创新，希望能够帮助读者建立更加清晰的思路与动机。

另外，多数信号与系统的教材对一些定理的边界情况的处理是含糊不清的，本书对于这些细节进行了严格的论证，并发现信号与系统教材中普遍存在的一些错误。尽管一般而言这些错误不会影响计算，但阅读本书可以获得更加深入的认识。
